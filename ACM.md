## 差分数组
差分数组的主要用途是**频繁对原始数组的某个区间的元素进行增减**

比如说有个数组`nums`,然后要求对区间`nums[2..6]`全部加一...  
第一种用for循环,但时间复杂度是O(N)  
第二种先对`nums`数组构造一个`diff`差分数组,`diff[i]`就是`nums[i]`和`nums[i-1]`之差  
```java
int[] diff = new int[nums.length];
//构造差分数组
diff[0] = nums[0];
for(int i = 1; i < nums.length; i++){
    diff[i] = nums[i] - nums[i-1];
}
```
通过这个代码可以反推出原始数组`nums`  
```java
int[] res = new int[diff.length];
//根据差分数组构造结果数组
res[0] = diff[0];
for (int i = 1; i < diff.length; i++){
res[i] = res[i-1] + diff[i];
}
```
如果要对数组`nums[i..j]`的元素全部加1,那么只需要让`diff[i] += 1`,然后再让`diff[j+1] -= 1`即可  
只要花费 O(1) 的时间修改 diff 数组，就相当于给 nums 的整个区间做了修改。多次修改 diff，然后通过 diff 数组反推，即可得到 nums 修改后的结果
```c++
//创建一个全为0数组，被移走就是加1，那么正的数就是被移走的，区间加减用差分
//变量定义
int s[100001],m,l,start,end,ans; //s差分数组
//main 函数里
cin >> l >> m;
for(int i = 1; i <= m; i++){
 cin >> start >> end;
 s[start]++,s[end + 1]--;
}
if(!s[0]) ans++;
for(int i = 1; i <= l; i++){
 s[i] += s[i-1];
 if(!s[i]){
   ans++;
 }
}
cout << ans;
```

## 异或  
`^`运算符,按位异或(XOR)  
相同为0,不同为1  
**特性**  
1. 任何数与0异或,结果还是本身  
2. 两个相同的数字异或,结果为0  
3. 满足结合律和交换律  

**`ans^=x`统计所有 x 值中，出现奇数次的数的异或结果**  
如果数`x`出现偶数次,那么异或后为0  
如果数`x`出现奇数次,那么异或后保留自身  
找出数组中出现奇数次的数（其他数都出现偶数次）使用异或运算可以高效解决这个问题，而且不需要额外的空间 

## 数字统计函数
```c++
int countDigitOne(int n) {
    int ones = 0;
    for (long long m = 1; m <= n; m *= 10) { // m: 1, 10, 100, 1000, ...
        int a = n / m;    // 当前位左边的部分（高位）
        int b = n % m;    // 当前位右边的部分（低位）
        ones += (a + 8) / 10 * m + (a % 10 == 1) * (b + 1);
    }
    return ones;
}
```  
`n/m`和`n%m`把数字`n`划分为两部分   
假设数字`554123`    
1. **第一种情况**      
分出的位大于1(k)   
分为`554`和`123`  
那么对于千位的前缀就是`0`到`55`  
`001000 ~ 001999 ->1000个`  
`011000 ~ 011999 ->1000个`  
`...`  
`551000 ~ 551999 ->1000个`  
`55199 < n`  
千位共出现有`(a/10 + 1) *1000` 
2. **第二种情况**  
分出的位等于1(k)  
分为`5541`和`23`  
那么对于百位的前缀就是`0`到`554`  
`000100 ~ 000199 -> 100个`  
...  
`554100 ~ 554123 ->只有24个`  
百位共出现有`(a/10)*100 + (b+1)`  
3. **第三种情况**  
分出的位小于1(k)  
共出现`(a/10)*100`  

对于这俩种可以合并  
当`a/10`的末尾是0或者1, 那么`a`最后一位加上8再除以10, 还是0  
       大于等于2时,`a+8`使得=`a/10`加一   
***所以***  
公式为 （（a + 8 ）/10 ）* m + (a %10 == 1)  * (b +1 )  
扩展 如果是数字k,那么a加`10-k-1`,三种情况不变
